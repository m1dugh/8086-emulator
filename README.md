# Intel 8086 Disassembler

## How to use ?

### Building

- Prod build
```bash
$ make
```

- Debug build
```bash
$ make debug
```

### Running

- Running prod binary
```bash
$ ./bin/Release/mmvm <file_to_disassemble>
```

- Running debug binary
```bash
$ ./bin/Debug/mmvm.debug <file_to_disassemble>
```
